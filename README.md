# Hide Photos Chrome Extension 🖼️

תוסף כרום שמסתיר תמונות בכל אתר ושומר רשימת אתרים בזיכרון המקומי.

## תכונות ✨

- **הסתרה מיידית** - הסתר את כל התמונות באתר בלחיצת כפתור אחת
- **רשימה חכמה** - שמור רשימה של אתרים שבהם הסתרת תמונות
- **זיכרון מקומי** - כל הנתונים נשמרים בטוח בזיכרון המקומי של הדפדפן
- **ממשק משתמש עברי** - ממשק נוח וידידותי בעברית
- **הסתרה כוללת** - מסתיר תמונות רגילות, תמונות רקע, ו-SVG

## מה מוסתר? 🙈

- `<img>` tags
- `<picture>` elements
- Background images
- תמונות שנטענות דינמית

## התקנה 🚀

1. פתח את Chrome וגש ל-`chrome://extensions/`
2. הפעל את **Developer Mode** (למעלה מימין)
3. לחץ על **Load unpacked**
4. בחר בתיקיה של התוסף

## שימוש 📖

1. כשאתה באתר שרוצה להסתיר בו תמונות, לחץ על אייקון התוסף
2. לחץ על **"הפעל הסתרה"**
3. התוסף יסתיר את כל התמונות ויוסיף את האתר לרשימה
4. להסרת אתר מהרשימה, לחץ על **"הסר"** ליד שם האתר
